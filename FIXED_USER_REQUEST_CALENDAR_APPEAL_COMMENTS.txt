Исправления по запросу:

1. Поле «Было ли вынесено определение кассационного суда?» показывается только при выборе «Кассация (ВС РФ)».
2. Автозапись календаря теперь называется:
   «Последний день подачи жалобы по делу № ..., Истец, Предмет спора».
3. Клик по календарной записи, связанной с общим перечнем дел, теперь переходит в «Общий перечень дел» и открывает полную карточку дела, а не мини-окно календаря.
4. Для комментариев на карточках добавлены пометки «Новый комментарий» / «Комментарий просмотрен»; при открытии карточки комментарий отмечается просмотренным.
